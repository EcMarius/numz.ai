OVH MODULE SETUP INSTRUCTIONS
===============================

Quick Setup Steps:
------------------

1. GET OVH API CREDENTIALS
   - Visit: https://eu.api.ovh.com/createApp/ (or your region)
   - Create application and get Application Key + Application Secret
   - Generate Consumer Key with permissions (see config.example.php)

2. INSTALL DATABASE TABLES
   - Run: php modules/servers/ovh/install.php
   - Or install via WHMCS addon modules interface

3. CONFIGURE WHMCS PRODUCT
   - Setup > Products/Services > Create Product
   - Module Settings tab:
     * Module: OVH Servers
     * Endpoint: ovh-eu / ovh-ca / ovh-us
     * Application Key: [your key]
     * Application Secret: [your secret]
     * Consumer Key: [your consumer key]
     * Service Type: dedicated / vps / cloud
     * Service Name/ID: [OVH service identifier]
     * Configure other options as needed

4. TEST CONNECTION
   - Go to product configuration
   - Click "Test Connection" button
   - Should return success with account details

5. SETUP CRON (OPTIONAL)
   - Add to system cron: php modules/servers/ovh/cron.php
   - Or access via: modules/servers/ovh/cron.php?token=your_token
   - Recommended: every 6 hours

6. ENABLE HOOKS (OPTIONAL)
   - Hooks are auto-loaded by WHMCS
   - Check System Settings > Module Settings > OVH

Required OVH API Permissions:
-----------------------------
GET/POST/PUT/DELETE on:
- /dedicated/server/*
- /vps/*
- /cloud/project/*
- /ip/*
- /me
