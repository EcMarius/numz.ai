# Tawk.to Live Chat Module for NUMZ.AI

This module integrates Tawk.to live chat into your NUMZ.AI installation.

## Installation

1. Enable the module from Admin > System Settings > Addon Modules
2. Enter your Tawk.to Property ID and Widget ID
3. Configure where you want the chat widget to appear

## Configuration

- **Property ID**: Found in your Tawk.to dashboard
- **Widget ID**: Found in your Tawk.to dashboard
- **Admin Area Only**: Show chat only in admin area
- **Client Area Only**: Show chat only in client area

## Features

- WHMCS compatible module structure
- Easy integration
- Configurable visibility
- Auto-injection of chat widget
